Pizza Sales Dashboard – Power BI

A Power BI dashboard project to analyze pizza sales and generate insights for data-driven decision-making.
The dataset contains order-wise pizza sales for one year.

Data Preparation (Power Query)
Steps performed:

Cleaned missing/invalid values
Converted data types (Date, Quantity, Unit Price)
Created calculated fields:
Total Revenue = Quantity × Unit Price

Business Outcomes

With this dashboard, the business can:

Identify top-selling pizzas
Track order patterns by day/month
Optimize inventory and marketing strategy
